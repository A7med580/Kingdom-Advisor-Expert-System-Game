# 👑 Kingdom Advisor — Expert System Game

An interactive web-based expert system game that demonstrates core **Intelligent Decision Support** concepts through a medieval kingdom management theme.

## 🎮 Live Demo

> Deploy to Vercel and your live link will appear here.

## 🧠 Expert System Concepts Demonstrated

| Concept | Description |
|---------|-------------|
| **Knowledge Base** | 18 IF-THEN production rules covering military, economy, people, and diplomacy |
| **Inference Engine** | Rule matching and conflict resolution engine |
| **Forward Chaining** | Data-driven reasoning from facts to conclusions |
| **Explanation Facility** | Step-by-step reasoning trace for every decision |

## 🚀 Deploy to Vercel (Free)

### Option 1: Via Vercel CLI
```bash
npm i -g vercel
cd expert-system-game
vercel
```

### Option 2: Via GitHub
1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"** → Import your repo
4. Framework: **Other** (static site)
5. Click **Deploy** — done!

### Option 3: Drag & Drop
1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag and drop the project folder
3. Deploy instantly

## 🛠️ Local Development

```bash
npx serve .
```

Then open `http://localhost:3000` in your browser.
